Акрополь

Компания “Акрополь” является одним из лидеров изготовления и продажи изделий из натурального гранита в Томской области. Основанная в 2010 году, компания уже более 10 лет успешно работает на внутреннем рынке России.

Задача:

Разработать дизайн, соответствующий тематике. Клиенту должно быть удобно использовать сайт, и ничего не должно его триггерить, человек даже в состоянии аффекта должен быть способен разобраться, как заказать услугу. Поэтому первоначальной задачей стало создать максимально понятный функционал.

Решение:

  1. Разработана система загрузки товара используя возможности фреймворк Django и языка python
  2. Верстка сайта по дизайн макету, и вывод динамической информации из админ панели используя шаблонизатор jinja2
  3. Оптимизированы orm запросы к базе данных
  4. Подготовлена кастомная админ панель для клиента

Использовались технологии:

Jinja

Django

HTML

SCSS

Figma
